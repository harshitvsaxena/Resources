# RethinkDB

## Installing RethinkDB

1. Install node first - link: https://nodejs.org/en/
2. Install rethinkDB - Download Link: https://www.rethinkdb.com/docs/install/

Now install client driver using:
```sh
$ npm install rethinkdb
```
